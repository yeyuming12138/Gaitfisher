# Gaitfisher
水产养殖场景步态数据集
1. 数据集简介
GaitFisher 是面向水产养殖场景的步态序列数据集，用于三种典型农事行为识别：死鱼捡拾（Dead-fish collecting），施药（Spraying），巡检（Inspection）。数据集由5名农事作业人员在真实养殖环境中录制，从室外养殖池塘收集了893个步态序列，从室内循环水养殖实验室收集了218个步态序列，总共1111个步态序列。其中，死鱼捡拾步态序列209例，施药步态序列116例，检查步态序列736例。
2. 数据内容与格式
- 原始数据：视频/图像序列（格式：.avi / .jpg）
- 解析数据：人体RGB/轮廓/解析
- 采集参数：分辨率1920×1080；帧率30 FPS；单序列长度3-4秒
- subject_id：'lcj','lst','scl','xxd','yym'
- scene：'inside','outside'
- action: 'collecting','spraying','inspection'
- sequence_id:'seq0','seq1'..
3. 数据开放状态
当前状态：暂未公开下载，为支持后续研究与可复现性，我们将在论文录用/后续项目完成后公开完整数据集与标注文件。
4. 推荐数据划分与评测协议
为便于可复现，我们将随数据发布：官方train/test划分文件（JSON/TXT），基准评测协议。
5. 隐私与伦理声明
数据采集已获得参与者知情同意。数据在发布前将进行必要的脱敏处理（如人脸模糊/去除可识别信息/仅发布预处理后数据），以降低个体可识别风险。
